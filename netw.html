Networking Fundamentals Workshop

7-Day Practical Learning on Network Infrastructure & Protocols

Day 1: Introduction to Networking Concepts

Date: 07.04.2025 | Duration: 1.5 hrs

  • What is a network? Types: LAN, WAN, MAN, PAN
  • Overview of OSI and TCP/IP models
  • Networking terminology and protocols
  • Discussion: Why networking matters in modern tech

Day 2: Networking Devices and Topologies

Date: 08.04.2025 | Duration: 1.5 hrs

  • Devices: Router, Switch, Hub, Modem, NIC
  • Common topologies: Star, Bus, Ring, Mesh
  • Hands-on: Identifying and cabling physical hardware
  • Discussion: Home vs Enterprise networks

Day 3: IP Addressing and Subnetting

Date: 10.04.2025 | Duration: 2 hrs

  • Understanding IPv4 and IPv6
  • Subnet masks, classes (A, B, C), CIDR notation
  • Practical: Subnetting exercises and IP planning
  • Static vs Dynamic addressing, DHCP overview

Day 4: Routing and Switching Fundamentals

Date: 12.04.2025 | Duration: 2 hrs

  • What is routing? Differences between routers and switches
  • Static and dynamic routing basics
  • VLANs, trunking, and inter-VLAN routing
  • Lab: Simulated packet routing using Cisco Packet Tracer

Day 5: Network Configuration and Troubleshooting

Date: 14.04.2025 | Duration: 1.5 hrs

  • Using CLI tools: ping, tracert, ipconfig, nslookup
  • Router and switch configuration basics
  • Troubleshooting common issues: IP conflicts, DNS, connectivity
  • Hands-on: Troubleshooting mini-lab

Day 6: Network Security and Best Practices

Date: 16.04.2025 | Duration: 1.5 hrs

  • Basics of network security: Firewalls, encryption, VPN
  • Securing wireless and wired networks
  • User access control, strong password policies
  • Real-world examples of network breaches and solutions

Day 7: Final Project, Review and Certification

Date: 17.04.2025 | Duration: 1.5 hrs

  • Review of all major topics
  • Group quiz and knowledge check
  • Feedback session and discussion on next steps (certifications, career)
  • Certification ceremony and closing remarks
